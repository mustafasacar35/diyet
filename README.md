
# Diyet Planlama Uygulaması

Bu proje, diyetisyenler ve hastalar için kapsamlı bir diyet takip ve planlama sistemidir.

## Özellikler
- Otomatik Diyet Planlama (Makro hedeflerine göre)
- Hasta Takip Portalı
- Akıllı Yemek Önerileri
- Detaylı Raporlama

## Teknoloji Yığını
- Next.js 14
- Supabase
- Tailwind CSS
- Shadcn UI

## Son Güncellemeler
- Yemek rolleri ve kategorileri standardize edildi.
- Vercel deploy entegrasyonu güncellendi.
